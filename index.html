Here's what's new in Scala

scala-cli will become the new scala command

Here's some cool things that scala-cli offers:

  • Can write your code as a script, no main, no @main, no nothing!
  • Totally remove the need for sbt! (dependencies are in comments at beginning of file)

Stop Coding in old style! scalac can automatically change your code

It seems that chatgpt, many templates (I'm talking to you sbt/g8), frameworks (I'm talking to you Play!) are stuck in scala 2.13 You can fix your source code (and chatgpt's suggestions with this:

scalac -new-syntax  -rewrite myfile.scala
 scalac -indent  -rewrite myfile.scala

The documentation is not too clear on this; I've found, like the docs, I had to use 2 steps, and it's not clear why of if there is another way
